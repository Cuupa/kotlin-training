== What did we just see?
[source, kotlin]
----
package kotlin_java.way2 #<1>

class Car { #<2>
    var licencePlate: String? = null #<3>
    var model: String? = null #<4>
    var manufacturer: String? = null
#<5>
}
----
<1> Kein Terminator (;)
<2> Kein Sichtbarkeitsmodifyier
<3> var?
<4> ??
<5> getter? setter?

== What did we just see? `visibility`

[source,kotlin]
----
class Car
----

[%step]
* `public` per default
** `public`, `private`, `package`, `internal`
* `final` per default
** `open`

== What did we just see? `var`
[source, kotlin]
----
 var licencePlate: String? = null
----

[%step]
* Schlüsselwort var
** "Normale" Variablendeklaration
* Schlüsselwort val
** Entspricht dem Java `final`
* `var`/`val` variablenname `:` Typ `=` ...

== What did we just see? `null`
[source, kotlin]
----
var model: String? = null
----

[%step]
* Kotlin ist nullsafe
* Nullable values werden mit `?` gekennzeichnet
* Kotlin ist nullsafe... except it isn't

[%step]
[source, kotlin]
----
var model: String? = null
var manufacturer: String = "Trabbi"
----
